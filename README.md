# Swiggy_Restaurant_Dashboard
Swiggy_Restaurant_Dashboard
# Project Objective.
The objective of this project is to analyze Swiggy restaurant data using Power BI 
and create an interactive dashboard to understand ratings, pricing, delivery time, popular food types,
and restaurant distribution across cities and areas.
# Key Insights (Important)
Insights:
1. The average restaurant rating is 3.66, which shows that most restaurants have moderate customer satisfaction.
2. The average food price is around ₹348, indicating the general pricing level of restaurants on the platform.
3. Indian food is the most served food type, followed by Chinese and North Indian cuisines.
4. Some areas like Vile Parle and Pacific Mall have the highest average food prices compared to other locations.
5. Kolkata and Mumbai have the highest number of restaurants among the analyzed cities.
